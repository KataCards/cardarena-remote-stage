Internal server error

The request reached the backend, but the service could not complete it successfully. This usually means the route is valid and the failure is happening inside the application.

What to check

  • Inspect the API or application logs for the failing request path.
  • Confirm required environment variables and secrets are present.
  • Retry after the service has recovered or after the deployment has been fixed.
